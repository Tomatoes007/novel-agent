# Novel Agent 开发清单

## 已完成
- 后端分层结构已搭建完成（core/models/schemas/repositories/services/agents/api/prompts）。
- FastAPI 主服务、根路由 `/`、健康检查 `/health` 已实现。
- 项目基础 CRUD 接口已实现（创建、列表、详情、更新）。
- 立项 Proposal 生成接口已实现：`POST /api/v1/agent/proposal`。
- 世界观生成与查询接口已实现。
- 角色生成与查询接口已实现。
- 大纲生成与查询接口已实现（卷纲、章节纲入库）。
- 章节生成主链路已实现（上下文加载、Prompt 组装、LLM 生成、入库）。
- 章节审校接口已实现（review）。
- 记忆回写已实现（章节摘要、伏笔、时间线、部分角色状态变化）。
- Redis 缓存已接入（最近章节摘要、上下文快照）。
- Milvus 向量检索已接入（写入向量、按 query 检索）。

## 待修改
- 增加统一异常处理与统一错误返回格式（当前多处直接抛异常或局部捕获）。
- 强化 JSON 解析稳定性（目前结构化输出解析偏脆弱）。
- 为 LLM/Embedding/Milvus 增加失败重试与降级策略。
- 完善检索策略（query 构造、chunk 粒度、召回后 rerank）。
- 优化章节生成 Prompt 与上下文拼装质量。
- 完善角色动态状态机（不仅限于 `power_changes`）。
- 完善伏笔生命周期状态（如 hinted/deepened/abandoned）。
- 增加世界观增量更新机制（而非仅初始化生成）。
- 修复并统一部分中文默认文案编码显示问题（出现乱码字符）。

## 待做
- 异步任务与队列机制（如 Celery/RQ/Dramatiq）。
- 任务状态跟踪与进度查询接口。
- 用户体系（注册/登录）与接口鉴权。
- 多租户与数据隔离能力。
- 权限控制（项目归属、访问控制）。
- 前端工作台（项目管理、角色卡、大纲树、章节编辑器、记忆可视化）。
- 更完整的审校能力（人设一致性、世界观冲突、节奏、钩子、爽点密度）。
- 导出能力（TXT/DOCX）与版本管理。
